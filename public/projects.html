Projects

This is a list of the programming projects I have done or been a part of. Each project was made using the programming languages and technologies shown in the "programming skills" page.

Uraohjain website

Website made with astro.js. Added strapi functionality for content management purposes. link to the website

HomestakerOS

I am contributing to the frontend part of the HomestakerOS open source project funded by Ethereum Foundation and ssv.network. link to the HomestakerOS github

Virittämö website

I updated the homepage for the Virittämö of city of Helsinki. I used Next.js, Azure and strapi in this project. Unfortunately, the virittämö project has since been closed and the website is no longer online.

Tilatikki

Classroom space managing system requested by Kasko Helsinki. The tilatikki was a proof of concept project that was succesfully tested with the teachers of jätkäsaari elementary school. I worked as a lead fullstack developer in this project. Main technologies used were React, Node.js, Express.js, MongoDB, REST. We gave away the project and the rights for the Kasko Helsinki.

Suomenlatu

The goal of the project was to create easier and intuitive UI for the Suomenlatu users. Main technologies used were Nuxt, Vue.js, Quasar, REST. The project was later passed on to CRM service for further development.

Instagram app "Pointsi"

This app was commissioned through school. Mostly done by myself, but some of the styling (css) was made by two other persons. Basic idea of the app was to take pictures for example at restaurants and then tag the place you took the photo at to earn some credits to spend. Video showcasing basic functionality

Soul Odyssey

Soul Odyssey is a single player mobile game. I was the lead developer of the project. The game won the Oulu Game Lab 2020 competition. The game had nearly 5000 downloads in Google Play before it's removal in late 2023. itch.io page of the game

Oulu environment game

Oulu environment game was commissioned by the city of Oulu. The game was aimed to teach children importancy of environmental issues. I was responsible for the programming of the game. We made the basic foundation of the game and there were talks that some other students will keep working on the project if needed
